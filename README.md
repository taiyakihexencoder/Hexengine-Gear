# Hexengine-Gear
勉強がてら自作したUnity用ライブラリを公開します。  
自己プロジェクト向けに作っているので、使用は自己責任でお願いします。

※ 詳しい使い方ページは現在作成中。

## インポート方法

1. パッケージマネージャからinstall package from git URL...
2. `https://github.com/taiyakihexencoder/Hexengine-Gear.git?path=（ライブラリ名）#（version）`を入力  
   例) `https://github.com/taiyakihexencoder/Hexengine-Gear.git?path=com.hexengine.gear#0.2.0`

## 対応言語について

* 以下の言語が用意されています。
    * 日本語 / JAPANESE
    * 英語 / ENGLISH

## ライブラリについて

### ◆ com.hexengine.gear

#### animation
* 3Dモデルのボーンアニメーションを管理するためのライブラリです。

#### parameters
* パラメーターをテーブルで編集し、バイナリとして出力するライブラリです。

### ◆ com.hexengine.gear.ecs

* Unity.Entities関連のユーティリティライブラリです。

### ◆ com.hexengine.gear.addressables

* Unity.Addressables関連のユーティリティライブラリです。
